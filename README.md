**Founded by:** Benedict Samwel Mfanga  
**Co-Founded by:** Roque Ricco DeSouza

**Website:** https://kijani-pulse-foundation.vercel.app



### 🌱 About the Foundation

KijaniPulse Network is a youth-powered initiative founded by **Benedict Samwel Mfanga** and co-founded by **Roque Ricco DeSouza**. 

Our mission is to mobilize students through education, innovation, and action to create real environmental impact.

**Motto:** "Where Green Ideas Become Real Impact"
